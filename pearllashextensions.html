Pearl Lash Extensions is an e-commerce website built with Wordpress, and hosted on e-host. The company services include certified eyelash trainings and products. When I started with this project I originally built it with AngularJS, and I used Firebase to store user information. The client needed a way for the user to be able to view courses the company offers, buy products, and pay for certified training classes. As the project progressed, the website became too complicated for the client to manage without the help of a developer. I decided in order to make the website more "client friendly" and have a simple way for the client to manage it, I had to recreate the website in Wordpress and to use built in plugins that Wordpress offers for e-commerce, Square Space, etc. One of the biggest challenges I had was working with Wordpress and e-host because I had little knowledge about these platforms and the website needed to be live asap.
