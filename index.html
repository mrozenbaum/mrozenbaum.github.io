Pearl Lash Extensionsmore_vert
https://pearllashextensions.com/ close

Is a SPA created with AngularJS for a client who's services include eyelash extension services. I used Firebase to save user information. Providing the user with the ability to create a profile and save their products or training information for a later time. Users can contact the company via a simple form, social links, or phone. I was given a strict 'style bible' to go by. Specific colors, styles, and company logos. I achiveved the clients needs using Bootstrap and CSS. In the end, I refactored and redesigned this project to be more "client friendly" by hard coding the code into a Wordpress theme and making it easier for the client to access and update on the fly. The final version of the website also has additional features. The website launched April 2017, and you can view it via the link provided, and you can see the project on my github.

Hi Karlmore_vert
https://www.hikarl.com/ close

Hi Karl’s mission is to make it extremely easy for users to discover last-minute events around them. Karl is an AI-powered virtual concierge who aggregates all events happening in an area. Based on a proprietary IP, Karl leverages the user’s listening habits, connecting to the user’s streaming systems, geographic data, and behavioral data to provide relevant concert recommendations for each user. In addition, to reduce the numbers of unsold tickets, Hi Karl allows promoters to push last-minute tickets directly to fans, at the exact time they are looking for an event.

Project is still under process. Please contact me for my Project Portfolio.
